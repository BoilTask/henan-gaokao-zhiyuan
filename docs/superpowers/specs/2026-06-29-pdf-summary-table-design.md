# PDF 导出汇总表格

**日期**: 2026-06-29
**状态**: 已批准

## 概述

在志愿草表 PDF 导出中，于现有志愿组详情之后新增一个汇总表格，方便快速浏览全部志愿的概览信息。

## 位置

汇总表格放在 PDF **最后面**——所有志愿组详情之后、页脚之前。

## 数据范围

- 仅包含 **非隐藏组**（`hidden !== true`）
- 按志愿序号顺序排列

## 表格结构

### 列定义

**组级列（前6列）**：
| 列 | 数据来源 | 说明 |
|----|---------|------|
| 序号 | 递增（仅非隐藏组） | 志愿填报顺序 |
| 策略 | `g.strategy` | 冲/稳/保 |
| 院校代码 | `g.schoolCode` | |
| 院校名称 | `g.schoolName` | |
| 组码 | `g.groupCode` | |
| 计划 | `g.planType` | 高校专项/国家专项/普通组等 |

**专业列（每组3对，每对3列 = 9列）**：
每对包含：序号、代码、专业名称

专业1-3放在第1行，专业4-6放在第2行。

### 行布局

- 每个志愿组最多占 **2行**
- 专业数 ≤ 3：仅1行（第1行有组信息+专业，无需第2行）
- 专业数 4-6：2行（第1行：组信息+专业1-3；第2行：仅专业4-6，组级列留空）
- 专业数为0的组理论上不会出现，但仍应处理（仅显示组信息，专业列全空）

### 视觉设计

| 元素 | 样式 |
|------|------|
| 表头 | 灰底（#F5F2EA），加粗，小号字体 |
| 组间分隔 | 粗线（border-top: 2px solid var(--ink)） |
| 策略着色 - 冲 | 行底色 #FBF4E6（暖金），策略文字色 var(--gold) |
| 策略着色 - 稳 | 行底色 #D7E8E5（蓝绿），策略文字色 var(--teal) |
| 策略着色 - 保 | 行底色 #DCEAD9（绿），策略文字色 var(--sage) |
| 交替底色 | 无策略的组使用交替底色：偶数序号的组底色 #F8F7F4 |
| 专业代码 | `var(--display)` 字体，金色（var(--gold)） |
| 空专业格 | 灰色 `—` 占位 |

### 分页控制

- 汇总表标题行不可与表格分离
- 表头（`<thead>`）在跨页时重复
- 每个志愿组的行使用 `page-break-inside: avoid`，不跨页截断
- 组间分隔线处是安全分页点

## 实现方案

### 修改范围

1. **`volunteer_form.html` — `renderViewDoc()` 函数**：在组详情循环后、页脚前，新增汇总表 HTML 生成逻辑（约 50 行）
2. **`volunteer_form.html` — CSS**：新增 `.vd-summary` 相关样式（约 40 行）

### 不变部分

- 现有组详情渲染逻辑
- `exportPDF()` / `exportImage()` / `exportPDFfallback()` 导出函数
- 数据模型和状态管理

### 技术要点

- 纯 HTML `<table>` 实现，浏览器原生打印可保持矢量文本
- jsPDF 备用方案通过 html2canvas 截图同样可用
- 利用 CSS `@media print` 控制打印样式
